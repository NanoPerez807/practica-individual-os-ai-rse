# XML analysis

This project analyzes open access scientific papers using XML files generated with Grobid.

## Scripts

- **figures.py**: counts figures and plots a bar chart.
- **keywords.py**: generates a word cloud from abstracts.
- **links.py**: extracts links from XML files.

## Installation

See the `README.md` for detailed installation instructions.

## Usage

Once installed, you can run the tests with:

```
python -m unittest discover tests
```
